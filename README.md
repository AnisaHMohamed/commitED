## Getting Started

1. Create the `.env` 
2. Update the .env file with your correct local information 
  - username: 
  - password:  
  - database: `commited`
3. Install dependencies: `npm i`
4. Run the server: `cd backend npm start`
5. Visit `http://localhost:3000/`

## Dependencies

- Node 10.x or above
- NPM 5.x or above
- PG 6.x
